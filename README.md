To create this in Excel:

Enter the data in cells A1:B6.
In cell D2, type: =SUM(D2:D35)
In cell B2, type: =COUNT(B2:B35)
In cell D2, type: =AVERAGE(D2:D35)
Press Enter after each formula.
MINIMUM AND MAXMIMUM VALUES
MIN: =MIN(D2:D35)
MAX: =MAX(D2:D35)
LOGICAL FUNCTION USING IF STATMENT
IF(D2:D35>=1$500,"High price")
IF(D2:D35<1$500,"Standard price")
CONDITIONL FUNCTION 
SUMIF(D2:D35,"Electronics")
COUNTIF(D2:D35,">$100)
TEXT FORMATING
LEFT(A2:A35,2)
RIGHT(A2:A35,2)
MID(A2:A35,4,6)
